# Documentacao

Esta pasta centraliza materiais de apoio do `PortfolioHUB`, como guias,
anotacoes, especificacoes e referencias dos projetos do repositorio.

Uso esperado:

- documentacao geral do portfolio
- guias de execucao e manutencao
- notas tecnicas e arquiteturais

Projetos que possuem documentacao propria podem manter arquivos adicionais
dentro de suas respectivas pastas.
