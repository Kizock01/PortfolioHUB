estruturas de controle e manipulacao de dados.

Arquivos atuais:

- `sistema-biblioteca.py`: simulacao de cadastro e busca de livros
- `limpeza-e-vetorizacao.py`: pipeline simples de limpeza de texto e indices
- `extrato-mensal.py`: processamento basico de receitas e despesas
